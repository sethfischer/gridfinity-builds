=================
Gridfinity builds
=================

Gridfintiy modules built with `michaelgale/cq-gridfinity <https://github.com/michaelgale/cq-gridfinity>`__.


Install
-------

.. code-block:: text

    poetry env use python3.9
    poetry install


Use
---

.. code-block:: text

    poetry shell
    invoke --list
    invoke task1 ... taskN
